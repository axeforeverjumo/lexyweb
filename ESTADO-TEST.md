# Estado del Proyecto - LexyApp

## Visión General
LexyApp es una aplicación de gestión de contratos legales que permite a las empresas organizar, buscar y analizar sus contratos de forma inteligente.

## Arquitectura Actual
- Frontend: Next.js + React
- Backend: Supabase
- AI: Claude API para análisis de contratos
